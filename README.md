# 🌾 India Agricultural Crop Production Analysis (1997–2021)

## 📖 Project Overview

This project analyzes India's agricultural crop production data from 1997 to 2021 using Tableau Public. The dashboard provides interactive visualizations to explore production trends, yield patterns, cultivated area, seasonal distribution, and state-wise agricultural performance.

---

## 🎯 Objectives

- Analyze agricultural production across Indian states.
- Study year-wise production trends.
- Compare agricultural yield over time.
- Identify leading agricultural states.
- Analyze cultivated area distribution.
- Understand seasonal agricultural production.
- Build an interactive Tableau dashboard for better decision-making.

---

## 🛠 Technologies Used

- Tableau Public
- Microsoft Excel
- GitHub

---

## 📊 Dashboard Features

- Total Production KPI
- Total Yield KPI
- State-wise Agricultural Production Map
- Year-wise Agricultural Production
- Agricultural Yield Trend
- State-wise Cultivated Area Comparison
- Production Distribution by Unit Type
- Leading Agricultural States
- Season-wise Agricultural Production
